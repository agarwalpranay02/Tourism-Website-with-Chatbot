# 🌍 Tourism Website with Chatbot Assistant

## 📌 Overview

This project is a responsive tourism website designed to showcase travel packages, adventure activities, and booking features. It includes an interactive chatbot assistant and a contact form for user queries.

---

## 🚀 Features

* Responsive UI for tourism website
* Travel packages and adventure sections
* Integrated chatbot for user assistance
* Contact form with backend database storage
* Smooth navigation and interactive design

---

## 🤖 Chatbot System

* Built using JavaScript + PHP
* Provides travel-related responses
* Handles user queries dynamically
* Enhances user engagement

---

## 📬 Contact Form

* User input stored using PHP + MySQL
* Form validation and async submission
* Real-time success/error feedback

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
